🌍 GEO GENIUS: WORLD QUIZ

About this Game

Geo Shapes is a fun geography learning game for kids. Can you recognize Italy by its boot shape? What about France or Japan?

Geo Shapes is a simple one-tap educational game that helps children learn country shapes in a fun and engaging way. No complex controls, no time pressure – just pure learning fun!

🎮 How to Play

  • A country outline appears on screen.
  • Tap the correct name from four options.
  • The fewer incorrect guesses, the more stars you earn.
  • Replay levels to get all 3 stars.

⭐ Features

  • 150+ countries to learn
  • Simple gameplay for young learners
  • Star reward system
  • Kid-friendly design
  • Works offline

👶 Target Age

Designed for children aged 12-15, but fun for the whole family.

📚 Educational Value

  • Recognize country shapes
  • Learn country names
  • Improve visual memory
  • Encourage self-improvement
